# Codon_Translation
> Translate single amino acid into nucleotides with command line. The purpose of the program is to identify synonymous mutation in a nucleotide sequence post sanger sequencing.

## Author
Ge Zhu < zhu.ge@husky.neu.edu>

Date: 02-10-2019
Update date: 1-15-2020


## Description
- Translate the codon into amino acid with the input file provided.
- Program know how to handle input that cannot be divided by three and stop codon.
- Calculate the average molecular weight with protein amino acid sequence.
